# Dev Setup

1. Clone Repo:

```
 $ git clone https://github.com/Sahaj001/snazzy-snowdrops.git
 $ cd snazzy-snowdrops
```

2. [Install Poetry](https://python-poetry.org/docs/#installation)
3. Run `poetry install --with dev`
4. Run `pre-commit install`

## Start dev setup

```bash
npm install
npm run dev
```
